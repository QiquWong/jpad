Class RotationOrder

java.lang.Object
  org.apache.commons.math3.geometry.euclidean.threed.RotationOrder

public final class RotationOrder
extends Object

This class is a utility representing a rotation order specification for Cardan or Euler angles specification. This class cannot be instanciated by the user. He can only use one of the twelve predefined supported orders as an argument to either the Rotation.Rotation(RotationOrder,double,double,double) constructor or the Rotation.getAngles(org.apache.commons.math3.geometry.euclidean.threed.RotationOrder) method.
